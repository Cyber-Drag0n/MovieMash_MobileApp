# Logic update

Что добавлено в Android Native:

- Добавлен отдельный экран выбора авторизации `auth` с двумя рабочими кнопками: `Войти` и `Регистрация`.
- В боковом меню для гостя теперь показывается пункт `Вход / Регистрация`.
- На главной быстрый пункт `Аккаунт` заменён на `Регистрация`, чтобы пользователь сразу попадал на auth-flow.
- Кнопка регистрации вызывает `POST /api/auth/register`, сохраняет JWT через `SessionManager` и переводит пользователя в аккаунт.
- Кнопка входа вызывает `POST /api/auth/login`, сохраняет JWT и переводит пользователя в аккаунт.
- Добавлено состояние загрузки для входа и регистрации, чтобы нельзя было отправить форму дважды.
- На странице фильма кнопка `Смотреть` теперь открывает поиск тайтла в VK Video.
- Добавлен блок `Где смотреть`:
  - кнопка `Смотреть в VK Video`;
  - кнопки провайдеров из TMDB `watch/providers`;
  - переходы формируются как в веб-версии: VK Video, Prime Video, Disney+, Netflix, Apple TV, Hulu, Paramount+, Peacock, Crunchyroll, MUBI, Plex, YouTube или Google fallback.
- Иконки на hero-карточке фильма теперь не декоративные:
  - закладка отправляет `POST /api/library/toggle` с `watchlist`;
  - сердце отправляет `POST /api/library/toggle` с `favorites`;
  - уведомления для гостя отправляют на auth-flow.
- Уже существующие действия сохранены:
  - отзывы отправляются через `POST /api/reviews`;
  - оценка отправляется через `POST /api/library/rate`;
  - поддержка отправляется через `POST /api/support/messages`;
  - профиль обновляется через `PATCH /api/account/profile`;
  - уведомления читаются через backend API.

Важно: для реальной записи в БД должен быть запущен backend на адресе из `app/build.gradle.kts` в `API_BASE_URL`.
