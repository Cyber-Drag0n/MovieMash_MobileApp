# MovieMash Native Android

## Что входит
- Jetpack Compose UI
- вход / регистрация / восстановление пароля
- экран главной
- поиск
- экран фильма/сериала
- аккаунт
- уведомления
- поддержка
- подписки
- работа с вашим backend API и TMDB

## Перед запуском
1. Откройте `app/build.gradle.kts`
2. Замените `REPLACE_WITH_TMDB_BEARER_TOKEN` на ваш TMDB bearer token
3. Проверьте `API_BASE_URL`
   - для эмулятора Android Studio локальный backend на ПК: `http://10.0.2.2:4000/`
   - для телефона укажите IP вашего компьютера в локальной сети
4. Запустите backend из папки `server/api` вашего старого проекта

## Как открыть в Android Studio
1. Распакуйте архив
2. Выберите `Open`
3. Укажите папку `MovieMashNative`
4. Дождитесь синхронизации Gradle
5. Запустите приложение на эмуляторе или устройстве
