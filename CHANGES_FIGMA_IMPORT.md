# MovieMash AndroidNative — правки по импортированному Figma-макету

Основа: экспорт `Movie App.zip` из Figma и нативный проект `MovieMash_AndroidNative.zip`.

## Что изменено

- Обновлена общая визуальная система под Figma:
  - фон `#141414`, карточки `#1A1A1A`, stroke `#262626`, акцент `#E50000`;
  - более жирная типографика для заголовков и CTA;
  - прямоугольные карточки с небольшим радиусом вместо слишком больших Material-скруглений.
- Переделан верхний мобильный header:
  - логотип MovieMash слева в стиле макета;
  - квадратная кнопка меню справа;
  - тёмный фон без стандартной Material TopAppBar-плашки.
- Главная страница приближена к мобильному Figma-экрану:
  - большой hero с постерной сеткой;
  - центрированный заголовок и описание;
  - красная CTA-кнопка `Начать просмотр`;
  - обновлённые секции и карточки.
- Экран поиска/каталога:
  - добавлен большой featured-блок в стиле Figma;
  - добавлен сегмент `Фильмы / Сериалы`;
  - поле поиска и кнопка приведены к стилю макета;
  - добавлен footer как в мобильном дизайне.
- Экран фильма/сериала:
  - верхняя hero-карточка с большим изображением, CTA и action-кнопками;
  - описание вынесено в отдельную тёмную карточку;
  - отзывы и служебные блоки переведены на новую палитру;
  - добавлен footer.
- Drawer-меню синхронизировано с новой палитрой.

## Ограничения проверки

APK не собран в этой среде: `gradlew` пытается скачать Gradle с `services.gradle.org`, а сетевой доступ к этому хосту недоступен. Проверка выполнена статически по исходникам; баланс скобок в основных Compose-файлах корректный.
