# Backend

Это backend вашего проекта MovieMash.

## Запуск
1. Скопируйте `.env.example` в `.env`
2. Заполните значения
3. Выполните `npm install`
4. Запустите `npm start`

## Для Android-эмулятора
В `.env` укажите:
- `CORS_ORIGIN=http://10.0.2.2:5173` или ваш фронтенд/тестовый адрес
- `FRONTEND_URL` при необходимости в основном серверном `.env` тоже должен указывать на правильный адрес, если используете восстановление пароля
